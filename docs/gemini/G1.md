Để chuộc lỗi, đây là đoạn code **Next.js (App Router)** chuẩn chỉ để bạn tích hợp vào **Antigravity IDE**. Mình sẽ dùng thư viện `rss-parser` vì nó cực kỳ ổn định để xử lý Feed từ Steam (App ID 730).

### 1. Cài đặt thư viện
Mở terminal trong Antigravity IDE và chạy:
```bash
npm install rss-parser
```

### 2. Tạo Service xử lý Steam RSS (`lib/steam-rss.ts`)
Đây là phần logic để bóc tách dữ liệu từ Steam. Mình sẽ thêm một chút "gia vị" lọc tin tức liên quan đến **Skins/Market** để phục vụ đúng chuyên môn của bạn.

```typescript
import Parser from 'rss-parser';

const parser = new Parser();
const STEAM_CS2_RSS = 'https://steamcommunity.com/games/730/rss/';

export async function getCS2News() {
  try {
    const feed = await parser.parseURL(STEAM_CS2_RSS);
    
    return feed.items.map(item => {
      // Logic đơn giản để tag các tin quan trọng về kinh tế/skins
      const content = item.contentSnippet || '';
      const isMarketUpdate = /case|collection|sticker|skin|armory|market/i.test(content);
      
      return {
        title: item.title,
        link: item.link,
        pubDate: item.pubDate,
        summary: content.slice(0, 200) + '...',
        isMarketUpdate
      };
    });
  } catch (error) {
    console.error('Lỗi lấy feed từ Steam:', error);
    return [];
  }
}
```

Để nội dung từ Steam RSS hiển thị chuyên nghiệp trên **Kanocs.com**, bạn cần xử lý các thẻ BBCode (đặc thù của Steam) và các thực thể HTML dư thừa. Nếu không, giao diện của bạn sẽ đầy những đoạn mã như `[img]`, `[b]`, hay `&nbsp;`.

Dưới đây là hàm `cleanSteamContent` tối ưu để bạn bỏ vào dự án Next.js trong **Antigravity IDE**:

### 1. Hàm xử lý nội dung (`lib/utils.ts`)

```typescript
export function cleanSteamContent(rawHtml: string): string {
  if (!rawHtml) return "";

  let cleaned = rawHtml
    // 1. Xử lý BBCode của Steam sang HTML cơ bản hoặc xóa bỏ
    .replace(/\[img\](.*?)\[\/img\]/gi, '') // Xóa ảnh để giữ layout news gọn (hoặc chuyển thành <img src="$1" />)
    .replace(/\[b\](.*?)\[\/b\]/gi, '<strong>$1</strong>')
    .replace(/\[u\](.*?)\[\/u\]/gi, '<u>$1</u>')
    .replace(/\[i\](.*?)\[\/i\]/gi, '<em>$1</em>')
    .replace(/\[url=(.*?)\](.*?)\[\/url\]/gi, '<a href="$1" class="text-blue-400 underline" target="_blank">$2</a>')
    
    // 2. Xóa các thẻ HTML rác thường đi kèm trong RSS của Steam
    .replace(/<br\s*\/?>/gi, ' ')
    .replace(/&nbsp;/g, ' ')
    .replace(/<(?:.|\n)*?>/gm, ' ') // Xóa tất cả các thẻ HTML còn sót lại nếu bạn chỉ muốn lấy text thô
    
    // 3. Làm sạch khoảng trắng thừa
    .replace(/\s+/g, ' ')
    .trim();

  return cleaned;
}
```

### 2. Cập nhật Service lấy Feed (`lib/steam-rss.ts`)

Kết hợp hàm `cleanSteamContent` vào logic lấy dữ liệu để Bot của **Kanocs** nhận được text "sạch" nhất phục vụ phân tích:

```typescript
import Parser from 'rss-parser';
import { cleanSteamContent } from './utils';

const parser = new Parser();

export async function getCS2News() {
  try {
    const feed = await parser.parseURL('https://steamcommunity.com/games/730/rss/');
    
    return feed.items.map(item => {
      const rawContent = item.content || item.contentSnippet || '';
      const cleanText = cleanSteamContent(rawContent);
      
      // Phân loại tin cho Kanocs
      const isUpdate = item.title?.toLowerCase().includes('update');
      const hasNewItems = /case|collection|charm|sticker/i.test(cleanText);

      return {
        title: item.title,
        link: item.link,
        pubDate: new Date(item.pubDate || '').toLocaleDateString('vi-VN'),
        description: cleanText.slice(0, 250) + '...', // Cắt ngắn để làm preview
        type: isUpdate ? 'Update' : 'News',
        marketImpact: hasNewItems ? 'High' : 'Low'
      };
    });
  } catch (error) {
    console.error('RSS Error:', error);
    return [];
  }
}
```

### Tại sao setup này "ngon" cho Kanocs?
1. **Dữ liệu chuẩn:** Loại bỏ hoàn toàn nhiễu từ BBCode, giúp Bot của bạn không bị đọc nhầm các ký tự mã nguồn.
2. **Cảnh báo sớm (Market Impact):** Tự động phát hiện các từ khóa về skin/case để đẩy lên đầu trang, giúp người dùng Kanocs phản ứng nhanh với giá thị trường.
3. **UI sạch sẽ:** Text được trim gọn gàng, không bị lỗi font hay ký tự lạ.
